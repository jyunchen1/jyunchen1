- **全端軟體工程** — 從前端介面、後端 API 到資料庫設計與部署 <br/> <sub>_Full-stack engineering — from frontend interfaces and backend APIs to database design and deployment._</sub>

### 🗂️ Selected work

- **StoneMark** — 自架的全端 Web 應用（`Next.js` · `Prisma` · `TypeScript`），涵蓋認證、資料模型與部署流程 <br/> <sub>_A self-hosted full-stack web application (`Next.js` · `Prisma` · `TypeScript`) covering authentication, data modeling, and deployment._</sub>
- **CairnPanel** — 基於 MCSManager 改造的應用程式管理面板，讓管理對象不再侷限於 Minecraft，各類服務與 App 都能在同一處統一控管 <br/> <sub>_An application management panel adapted from MCSManager — extending it beyond Minecraft so any service or app can be managed from a single place._</sub>
- **石家庄 Minecraft Server** — 由築石工作室營運的 Minecraft 伺服器，涵蓋插件開發（`Java`）、伺服器維運與玩家系統 <br/> <sub>_A Minecraft server operated by Cairn Studio — Java plugin development, server operations, and player systems._</sub>
